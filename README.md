# esp32_dab_radio

This is a project for an ESP 32 based audio sytem with FM & DAB+ radio support as well as Bluetooth.

The target is to renovate a vintage [Philips radio (model BF501)](https://location.hifivintage.eu/en/tubes-radio/3175-location-PHILIP-BF-501-A.html).
